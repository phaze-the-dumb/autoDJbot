# autoDJbot

plays a folder of mp3 files on repeat forever until you turn it off. has `!skip` and `!volume <number (0 - 100)>` commands. thats litterally it.

clone the repo...

run `npm i`

open index.js

change these

```js
let path = 'Path to folder of .mp3 files'
let TOKEN = 'Bot token'
```

run `node .`
